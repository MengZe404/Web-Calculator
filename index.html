Tutorial

Features

What can you use?

  • Mode selector: Standard (default), Algebra & Calculus.
  • Input box: input the mathematical expression you want the calculator to evaluate.
  • Output box: display your input expression & result.

Calculator modes

The calculator modes in detail.

Standard: basic mathematical evaluations.
  • Arithmetic calculations (+,-,*,^,/,%, sqrt)
  • Advanced calculations:
    • log(a,b): returns n where b^n = a.
    • trigonometry [sin(),cos(),tan(),sec(),csc(), cot()]
  • Unit converter (e.g 10km/hour to m/s)
Algebra: mathematical calculations involving functions.
  1. Define function (e.g f(x) = 10x) [case sensitive!]
  2. Input x and get result
  3. Move on to the next function, define g(x) or h(x) if f(x) is used.
Calculus: mathematical calculations involving differentiation.

Input --> 5x^2 + 10x - 5
Output <-- 10x + 10
